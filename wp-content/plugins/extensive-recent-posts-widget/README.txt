== Description ==

Recent posts widget with fully customizable layout and advanced options.

== Installation ==

1. Download and unzip plugin to `/wp-content/plugins/` directory
2. Activate plugin
3. Go to `Appearance -> Widgets` and drag'n'drop Extensive Recent Posts Widget to needed sidebar
